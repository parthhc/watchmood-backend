# WatchMood backend

The backend for an multi-agent workflow to recommend movies. Frontend repo [here](https://github.com/parthhc/watchmood-frontend), will need both to run.

## Features

- Movie recommendations based on a short prompt. Takes advantage of multiple tools to make the workflow agentic.
- Vector DB storage for movie overviews and reviews to be used as a tool for the agent to use queries to generate recommendations
- A custom taste profile generated by an AI agent after every fifth review, with access to tools to query the movie via the TMDb API and to get recent reviews.
- CRUD functionality for reviewing movies

## Tech Stack

- Pydantic AI — agent framework + structured outputs
- ChromaDB — two collections (movie overviews, user reviews)
- TMDB API — live ratings, metadata, search
- FastAPI - API for frontend

## Set Up Commands

1. Please have `uv` installed. See docs or use the following command

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. Set up your .env. Bring your own keys for TMDb and Gemini

```
cp .env.sample .env
```

3. Install deps

```
uv sync
```

4. Run DB seeder

```
uv run src/scripts/seed_movies.py
```

## Run command for the app

```
uvicorn main:app --reload
```
